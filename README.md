Overview :-

Power BI dashboard project designed to analyze business performance by tracking key metrics such as profit, loss, demand, availability, and supply shortages.The goal of this project is to provide actionable insights that help in better decision-making and operational efficiency.
Microsoft SQL Server and MYSQL Database as Datasource for this project. A Power BI report using Test Environment in SQL Server and then shift the report from Test Environmnent to Production Environment. Transaction all your report from SQL Server as a Datasource to using MYSQL database as the datasource.

Objectives :-

1)Analyze overall profit and loss trends.
2)Monitor daily demand vs availability.
3)Identify supply shortages.
4)Enable data-driven business optimization.

Key Insights :-

1)Demand is significantly higher than availability → leads to shortages.
2)High total loss compared to profit → requires cost optimization.
3)Daily loss trend indicates continuous operational inefficiencies.



Tools & Technologies Used :-

Power BI Desktop
Data Modeling & DAX
Data Cleaning & Transformation (Power Query)
Interactive Visualizations
